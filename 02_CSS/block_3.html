Блок навигации

  • Ширина 15 em
  • Должен присутствовать цвет фона
  • заспологается слева
  • Блок наполнить контентом достаточным, чтобы по высоте он начинал захватывать вторую статья, как показано в данном примере

Статья 1

Размер блока:

Занимает всю доступную ширину

Должен присутствовать цвет фона

Во всех "статьях" сделать отступ снизу от соседей равным 2em, при помощи свойства margin-bottom

Статья 2

Размер блока:

Занимает всю доступную ширину

Должен присутствовать цвет фона

Во всех "статьях" сделать отступ снизу от соседей равным 2em, при помощи свойства margin-bottom. Картинка вставлена при помощи тега, плавание организованно по правому краю внутри данной статьи для этой картинки

Нижний блок, подвал. Занимает всю доступную ширину. Выста определяется помещённым внутрь контейнером.
